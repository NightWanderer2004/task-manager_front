# Frontend for a task-manager app

### Technologies used:

-  React
-  Apollo Client
-  GraphQL
-  TypeScript
-  Material UI
-  Formik

### How to run:

-  Clone the repository
-  Run `npm install` to install dependencies
-  Run `npm start` to start the app
-  Open `http://localhost:3000` to view it in the browser.
